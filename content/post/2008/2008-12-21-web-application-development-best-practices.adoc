. Follow consistency in naming the project artifacts like Java class names, JSP names etc.
. Maintain the standard way of handling exceptions in different layers.
. Have a single access point for Configuration Constants.
. Take care about page navigation. Use forwarding and redirecting carefully.
. Always try to pull out the repeated code blocks and make them separate methods.
