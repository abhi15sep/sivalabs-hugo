What is an Interface?::  
  An Interface is a contract between the Caller and Called Objects.  
  An Interface only declares the signatures of the methods that the caller object can invoke on 
  the object that implements the Interface.  
  
What is a Class?::  
  A Class is a blueprint which represents the structure and behavior of a set of same kind of Objects.  
  A Class is a template defining the attributes and methods that are shared by a set of same kind of Objects.
